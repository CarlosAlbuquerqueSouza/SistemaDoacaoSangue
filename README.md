# 🩸 Sistema de Controle de Doação de Sangue

Projeto de Extensão Interdisciplinar IV — área de **Saúde e Acessibilidade**.

---

## Sobre

Sistema desenvolvido para controlar o estoque de bolsas de sangue coletadas nos Hemocentros, oferecendo uma interface mais acessível e intuitiva do que as soluções existentes. O foco está em democratizar o acesso às informações de estoque e notificações para usuários com diferentes níveis de familiaridade tecnológica.

---

## Funcionalidades

- Cadastro e gerenciamento de doadores
- Controle de estoque por tipo sanguíneo
- Agendamento de doações
- Monitoramento dos níveis de estoque em tempo real
- Notificações e alertas sobre necessidade de doações
- Interface acessível para todos os perfis de usuário

---

## Público-Alvo

Pessoas entre **16 e 69 anos** que atendam aos requisitos para doação de sangue:

- Peso mínimo de 50kg
- Boa saúde no momento da doação
- Menores de idade com autorização; primeira doação até os 60 anos
- Respeito ao intervalo entre doações: **4 meses** para mulheres / **3 meses** para homens
